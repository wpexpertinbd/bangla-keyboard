Bangla Keyboard Layout for Mac

Version __VERSION__ · by BiswasHost · www.biswashost.com

Installs two fixed Bangla keyboard layouts and a set of free Unicode Bangla fonts.

  • Bangla Unicode & Bangla Classic → /Library/Keyboard Layouts
  • 14 free (GPL / SIL OFL) Unicode Bangla fonts → /Library/Fonts
  • Standard shortcuts keep working — ⌘C / ⌘V / ⌘X and all ⌘ commands fire normally while a Bangla layout is active.

After installing: System Settings → Keyboard → Text Input → Edit… → + → Bangla → add the layouts, then log out and back in.

Note: the Bangla Classic layout outputs the legacy ASCII (non-Unicode) encoding used by legacy ANSI (“MJ”-style) Bangla fonts. Those fonts are not included (they are proprietary) — install a compatible legacy ANSI Bangla font from your own source to use the Classic layout. The Bangla Unicode layout works with any of the bundled Unicode fonts.

Apple Silicon & Intel, macOS 11+. No Java required. MIT-licensed.
