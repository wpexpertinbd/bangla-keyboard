This app is FREE, open-source and safe -- but it is NOT signed by Apple
(that requires a paid Apple Developer account). So the FIRST time you open
it, macOS shows a security warning. This is normal. Here is exactly what
to do.


------------------------------------------------------------
HOW TO INSTALL  (English)
------------------------------------------------------------

1. Double-click  "Bangla Keyboard Installer".

2. macOS shows:  "Bangla Keyboard Installer" Not Opened --
   "Apple could not verify ...".   Click  Done.

3. Open the  Apple menu  ->  System Settings  ->  Privacy & Security.

4. Scroll down to the  "Security"  section. You will see:

        "Bangla Keyboard Installer" was blocked ...   [ Open Anyway ]

   Click  Open Anyway.

5. On the confirmation, click  Open Anyway  again, then unlock with your
   Touch ID  or  password.

6. The installer opens  ->  click  Install  ->  enter your password.

7. IMPORTANT:  Log out and log back in  (or restart your Mac).
   macOS only loads new keyboard layouts after a fresh login.

8. System Settings -> Keyboard -> Text Input -> Edit... -> "+" -> Bangla
   ->  add  "Bangla Unicode"  (and/or  "Bangla Classic").

9. Switch keyboards with the menu-bar flag icon, or press  Control + Space.

To uninstall later:  open this app again -> choose  Uninstall.

TIP: An even simpler installer, "Bangla Keyboard.pkg", is on the GitHub
Releases page. Right-click it -> Open -> Open  (no Settings step needed).

------------------------------------------------------------

অ্যাপটি ফ্রি ও নিরাপদ, কিন্তু Apple-এর সাইন করা নয় (সেটার জন্য Apple-এর
পেইড অ্যাকাউন্ট লাগে)। তাই প্রথমবার ম্যাক একটা সিকিউরিটি ওয়ার্নিং দেখাবে।
এটা স্বাভাবিক। যা করবেন:

১।  "Bangla Keyboard Installer"-এ ডাবল-ক্লিক করুন।

২।  ম্যাক দেখাবে: "Bangla Keyboard Installer" Not Opened --
    "Apple could not verify ..."।   Done চাপুন।

৩।  Apple menu  ->  System Settings  ->  Privacy & Security  খুলুন।

৪।  নিচে  "Security"  অংশে দেখবেন:

        "Bangla Keyboard Installer" was blocked ...   [ Open Anyway ]

    Open Anyway  চাপুন।

৫।  আবার  Open Anyway  চাপুন, তারপর  Touch ID  বা  পাসওয়ার্ড  দিন।

৬।  ইনস্টলার খুলবে  ->  Install  চাপুন  ->  পাসওয়ার্ড দিন।

৭।  জরুরি:  একবার লগ-আউট করে আবার লগ-ইন করুন  (বা রিস্টার্ট দিন)।
    নতুন কীবোর্ড লেআউট ম্যাক শুধু নতুন লগইনের পরেই দেখায়।

৮।  System Settings -> Keyboard -> Text Input -> Edit... -> "+" -> Bangla
    থেকে  "Bangla Unicode"  (ও/বা  "Bangla Classic")  যোগ করুন।

৯।  কীবোর্ড বদলান মেনুবারের পতাকা আইকন দিয়ে, অথবা  Control + Space  চাপুন।

পরে আনইনস্টল করতে:  এই অ্যাপটি আবার খুলে  Uninstall  বেছে নিন।

টিপ: আরও সহজ ইনস্টলার  "Bangla Keyboard.pkg"  GitHub Releases পেজে আছে।
ওটায়  right-click -> Open -> Open  করলেই হবে (Settings-এ যেতে হবে না)।

------------------------------------------------------------
TYPING TIP - Bangla Classic (ও-কার / এ-কার)
------------------------------------------------------------
Type the consonant FIRST, then the vowel sign - NO space:
   স then ে        -> সে
   স then ে then া -> সো
   ত then ে then া -> তো
This keeps Space as a clean word-gap, so words no longer stick
together.  (Older builds used Space to make these - just drop it.)
